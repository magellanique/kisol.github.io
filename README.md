# KISoL - A Standard of Living Index by magellanique.com

Static GitHub Pages site for the July 2026 KISoL results.

## Pages

- `index.html` - rankings, maps, search, project overview, and research-contact links
- `methodology.html` - public methodology and formulas
- `faq.html` - design choices, interpretation, and limitations
- `calculator.html` - salary and purchasing-capacity comparison tool
- `your-city.html` - custom city basket and affordability calculator

No Excel workbooks or internal research files are included in this publishable folder.
